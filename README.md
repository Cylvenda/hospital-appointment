# Frontend (Next.js)

This folder contains the Next.js frontend for the appointment management system.

## Stack

- Next.js (App Router)
- React
- TypeScript
- Tailwind CSS / shadcn setup

## Run Locally

```bash
cd frontend
npm install
npm run dev
```

Frontend URL: `http://localhost:3000`

## Scripts

- `npm run dev` - start development server
- `npm run build` - production build
- `npm run start` - run production build
- `npm run lint` - run ESLint

## Notes

- Keep backend running on `http://127.0.0.1:8000` for local API integration.
- UI theme variables are defined in `src/app/globals.css`.
